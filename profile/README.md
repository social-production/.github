# Social Production

Build production for use, not profit.

Social Production is an open source platform for organizing collective production, governance, funding, and distribution outside market relations.

## What We Are Building

- project coordination across all spheres of production; things like food, housing, care, tools, and free software
- collective governance recorded on-chain
- pooled resources without private ownership claims
- need-based distribution planning
- peer-to-peer infrastructure with local-first clients
